# Fly.io Valid Regions

**Common Regions:**
- `ord` - Chicago, USA
- `iad` - Washington, D.C., USA
- `dfw` - Dallas, USA
- `sea` - Seattle, USA
- `lax` - Los Angeles, USA
- `sjc` - San Jose, USA
- `sjc` - San Francisco Bay Area, USA (if available)
- `ams` - Amsterdam, Netherlands
- `cdg` - Paris, France
- `fra` - Frankfurt, Germany
- `lhr` - London, UK
- `nrt` - Tokyo, Japan
- `syd` - Sydney, Australia

**To see all valid regions:**
```bash
fly platform regions
```

**Recommendation for US:**
- `ord` (Chicago) - Central US, good latency
- `iad` (Washington DC) - East Coast
- `sjc` (San Jose) - West Coast (closest to SF)

**Updated fly.toml uses:** `ord` (Chicago)
